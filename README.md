# ⚛️ GARDIEN QUANTIQUE - Simulation d'authentification post-quantique

Un projet pédagogique qui simule les principes de la **cryptographie post-quantique** et de l'**intrication quantique** pour créer un système d'authentification innovant.

**🎯 OBJECTIF PÉDAGOGIQUE :** Ce projet démontre ma compréhension des concepts de base de la cryptographie post-quantique, du principe d'intrication quantique (vulgarisation), de la création d'un système d'authentification sécurisé et de l'innovation en cybersécurité.

**🔬 COMMENT ÇA MARCHE ?** Le programme simule des concepts quantiques : Superposition (génération d'un état à partir de nombres aléatoires cryptographiques), Intrication (lien entre l'utilisateur et son mot de passe via un identifiant unique), Principe d'incertitude (détection des "espionnages" par vérification d'intégrité), Non-clonage (auto-destruction du mot de passe en cas d'intrusion). ⚠️ Note importante : Ceci est une simulation pédagogique, pas un vrai système quantique.

**🚀 UTILISATION :** Installation : `python quantique.py`. Exemple : choix 1 → entrez votre nom "Test" → mot de passe généré "a3f5c2e1b8d4f7h9" → choix 2 → entrez votre nom "Test" → entrez le mot de passe "a3f5c2e1b8d4f7h9" → connexion réussie.

**📋 FONCTIONNALITÉS :** Option 1 = Créer un compte (génération d'un mot de passe unique), Option 2 = Se connecter (vérification), Option 3 = Afficher le niveau de sécurité simulé, Option 4 = Informations sur le projet, Option 5 = Quitter.

**🛠️ TECHNOLOGIES UTILISÉES :** Python 3, hashlib (SHA-3 pour la génération sécurisée), secrets (génération cryptographique), JSON (sauvegarde).

**📊 CE QUE CE PROJET DÉMONTRE :** Compréhension des concepts quantiques ⭐⭐⭐, Algorithmes cryptographiques ⭐⭐⭐, Programmation Python ⭐⭐⭐, Innovation et créativité ⭐⭐⭐⭐, Vulgarisation scientifique ⭐⭐⭐⭐.
