# ⚛️ GARDIEN QUANTIQUE

## Simulation d'authentification post-quantique

Un projet pédagogique qui simule les principes de la **cryptographie post-quantique** et de l'**intrication quantique** pour créer un système d'authentification innovant.

---

## 🎯 OBJECTIF PÉDAGOGIQUE

Ce projet démontre ma compréhension de :
- Les concepts de base de la cryptographie post-quantique
- Le principe d'intrication quantique (vulgarisation)
- La création d'un système d'authentification sécurisé
- L'innovation en cybersécurité

---

## 🔬 COMMENT ÇA MARCHE ?

Le programme simule des concepts quantiques :

| Concept quantique | Simulation dans le code |
|-------------------|------------------------|
| Superposition | Génération d'un état à partir de nombres aléatoires cryptographiques |
| Intrication | Lien entre l'utilisateur et son mot de passe via un identifiant unique |
| Principe d'incertitude | Détection des "espionnages" par vérification d'intégrité |
| Non-clonage | Auto-destruction du mot de passe en cas d'intrusion |

> ⚠️ **Note importante** : Ceci est une **simulation pédagogique**, pas un vrai système quantique.
